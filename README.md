# CV-Simon
## Overview

This repository contains my professional CV/resume.

## Format

The CV is available in multiple formats:
- PDF
- Markdown
- HTML

## Building

To generate the CV from source:

```bash
# Install dependencies
npm install

# Build all formats
npm run build
```

## Structure

```
.
├── src/           # Source files
├── output/        # Generated CV files
└── assets/        # Images and resources
```
